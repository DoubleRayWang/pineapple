# 菠萝监控系统
![logo](https://github.com/pwstrick/pineapple/blob/master/client/public/img/logo.png)

一套简易的前端监控系统，技术栈是：React+MongoDB+Node.js+Koa2。
```
├── pingapple --------------------------------- 菠萝监控系统
│   ├── client -------------------------------- 系统的前端部分
│   ├── sdk ----------------------------------- 信息搜集代码库
│   ├── server -------------------------------- 系统的后端部分
```
