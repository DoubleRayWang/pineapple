## 菠萝监控系统的后端部分

```
├── server --------------------------------- 服务端
│   ├── bin -------------------------------- 命令
│   ├── config ----------------------------- 配置目录
│   ├── controllers ------------------------ MVC中的逻辑层
│   ├── db --------------------------------- MVC中的数据层
│   ├── public ----------------------------- 静态资源
│   ├── routes ----------------------------- 路由
│   ├── views ------------------------------ MVC中的视图层
│   ├── app.js ----------------------------- 入口文件
```

## 开启服务
运行命令开启服务，支持热更新。
```
npm run dev
```
