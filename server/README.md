## 菠萝监控系统的后端部分

```
├── server --------------------------------- 服务端
│   ├── bin -------------------------------- 命令
│   ├── config ----------------------------- 配置目录
│   ├── controllers ------------------------ MVC中的逻辑层
│   ├── db --------------------------------- MVC中的数据层
│   ├── public ----------------------------- 静态资源
│   ├── routes ----------------------------- 路由
│   ├── utils ------------------------------ 工具库
│   ├── views ------------------------------ MVC中的视图层
│   ├── app.js ----------------------------- 入口文件
```

## 开启服务
运行命令开启服务，可访问的地址是`http://127.0.0.1:3001`，支持热更新。
```
npm run dev
```

## 添加测试数据
运行命令添加测试数据，项目名称为“测试项目”，性能参数和错误数据的创建日期从当前时间开始算起，每隔30秒添加一条。
```
npm run data
```
